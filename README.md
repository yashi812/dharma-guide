# 🕉️ Dharma Guide

A Flutter spiritual companion app rooted in Bhagavad Gita wisdom.

## Features
- Daily Gita verse
- AI-powered spiritual guidance
- Mantra puja practice with TTS
- Mood reflection journal
- Premium paywall

## Setup
1. Clone the repo
```bash
   git clone https://github.com/YOUR_USERNAME/dharma-guide.git
```
2. Install dependencies
```bash
   flutter pub get
```
3. Run the app
```bash
   flutter run
```

## Tech Stack
- Flutter / Dart
- Anthropic Claude API
- flutter_tts, speech_to_text
